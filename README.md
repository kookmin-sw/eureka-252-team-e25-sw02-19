### 팀 슬로건

Error 404: Limits Not Found

### 팀 소개

저희 팀의 슬로건인 'Error 404: Limits Not Found'는 웹에서 '페이지를 찾을 수 없다'는 의미의 404 에러 코드를 재치 있게 활용한 것입니다.

보통 404 에러는 한계에 부딪혔을 때 나타나지만, 저희 19조에게는 한계(Limits) 자체가 존재하지 않는다는 당찬 포부를 담고 있습니다.
